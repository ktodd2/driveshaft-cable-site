# App.jsx

## Purpose

Root application component that orchestrates the entire single-page marketing website for the K.Todd Driveshaft Cable product. Composes all major page sections in a logical flow from header through footer, establishing the page structure and dark theme container.

## Exports

- `App` (default) - Main React component that renders the complete page layout with all section components

## Dependencies

- react - Core React library for component rendering
- [[src-components-header]] - Fixed navigation header component
- [[src-components-hero]] - Hero section with product introduction
- [[src-components-problem]] - Problem statement section
- [[src-components-product]] - Product solution/benefits section
- [[src-components-specs]] - Technical specifications section
- [[src-components-howitworks]] - Installation instructions section
- [[src-components-customers]] - Target customer segments section
- [[src-components-quoteform]] - Lead capture form section
- [[src-components-footer]] - Site footer component
