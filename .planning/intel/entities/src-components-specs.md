# Specs.jsx

## Purpose

Technical specifications section displaying detailed product measurements and materials. Renders a grid of six specification cards (cable diameter, length, working load, end construction, coupler material, cable material) plus a list of eight supported vehicle/equipment types. Includes a safety warning about proper usage.

- `Specs` (default) - Specifications section React component
