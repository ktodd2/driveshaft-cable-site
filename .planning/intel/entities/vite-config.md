# vite.config.js

## Purpose

Vite build tool configuration for the React application. Enables the React plugin for JSX transformation, sets the development server port to 3000, and configures the production build output directory to 'dist'.

## Exports

- `default` - Vite configuration object

## Dependencies

- vite - Build tool and dev server (defineConfig helper)
- @vitejs/plugin-react - Vite plugin for React support
