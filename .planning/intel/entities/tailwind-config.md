# tailwind.config.js

## Purpose

Tailwind CSS configuration defining custom theme extensions for the K.Todd brand. Specifies content paths for class scanning, extends the color palette with brand colors (ktodd-yellow, ktodd-gold, ktodd-steel, ktodd-dark, ktodd-charcoal), and defines custom font families for industrial and body text styles.

## Exports

- `default` - Tailwind CSS configuration object

## Dependencies

- tailwindcss - Utility-first CSS framework (configuration schema)
